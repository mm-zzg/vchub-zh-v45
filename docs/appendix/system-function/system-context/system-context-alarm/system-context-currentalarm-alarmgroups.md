

# System.Context.currentAlarm.alarmGroups


## Description
The alarm group(s) where the current alarm is located.

## Grammar
System.Context.currentAlarm.alarmGroups

## Code Example                                                                                                                  
Get the alarm group(s) where the current alarm is located.

```typescript 

const groups = System.Context.currentAlarm.alarmGroups;
console.log(groups);

```

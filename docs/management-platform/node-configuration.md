# 节点配置

节点配置页面主要用来配置节点的名称，web服务器的端口等。

![alt text](12.png)

| **名称**   | **描述**|
|:------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 节点名称   | 当前节点的名称，组网时使用节点名称作为节点的唯一标识。名称中使用的字母区分大小写。如果当前节点  启用了冗余，主备节点的节点名称需要保持一致，冗余节点的节点名称需要手动修改成主节点的节点名称。<br>**注意**：节点形成组网或者冗余后，不建议修改节点名称，会影响历史数据的采集。 |
| 服务器语言 | VC Hub程序的服务端语言。|
| HTTP端口   | 默认8066，支持修改，配置web服务的HTTP端口。|
| HTTPS端口  | 默认10043，支持修改，配置web服务的HTTPS端口。|
| 公共地址   | 面向外部网络的网络地址，主要用于运行时画面主备切换。|

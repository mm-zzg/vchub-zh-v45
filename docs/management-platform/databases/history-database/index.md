# 历史库

历史库模块为存储和访问历史数据提供了强大的功能和灵活性。在WAGO VC Hub上启用变量历史记录后，数据会以有效的格式自动存储在数据库中。此数据可用于图表和报表查询。用于分区和删除旧数据的选项有助于确保系统以最少的额外工作保持维护。

#### 支持的历史库类型

1. **历史库**：对存储到[数据库连接](../database-connection/index.md)配置中的数据进行分区、预处理、数据清理等配置。
2. **双历史库**：数据会同时存储到配置的数据库连接1和数据库连接2中，对历史数据的存储和查询进行灾备处理。当数据库连接1失效后，将切换到数据库连接2，继续提供存储和查询功能。
3. **远端历史库**：对组网内其他节点的历史库、双历史库进行转发配置，使用此配置时，数据存储和查询将使用远程节点的历史库配置。

# 集成接口定义

#### GET /api/v1/scripts/execute

执行用户自定义函数，并返回自定义函数的执行结果

协议: Https

请求负载(Json):

| 名称       | 类型      | 描述     |
|:------------|:-----------|:-----------------------------------------------------------------|
| scriptName | String    | 自定义函数名称     |
| parameters | dynamic[] | 自定义函数的参数列表，参数的类型必须与实际系统函数的参数类型一致 |

响应负载(JsonObject): 自定义函数的执行结果

# 3D建模

本章节主要介绍3D模型的制作标准，所用的建模软件默认为 **3DMAX**。

如果您具备建模能力，并且希望将创建的模型导入到WAGO VC Hub中，请按照此文档中的相关要求和标准创建模型。

## 模型制作流程

#### 准备工作

1.  提供模型高清照片，模型照片需要前后左右视图，并且能看清模型细节和结构信息。
2.  模型实际尺寸比例图。

#### 建模标准及注意事项

1. 软件单位设置。
2. 模型中心轴设置。
3. 模型制作对面的要求和控制。
4. 比例正确和简单效果。
5. 注意事项。（主要为建筑模型）
6. 模型最终检查。 （ 检查部分包含：废点、废线、废面、 漏面漏缝、多边面、反转面问题）

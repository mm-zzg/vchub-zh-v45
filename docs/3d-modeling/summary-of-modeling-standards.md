# 建模标准总结

#### 软件要求

1.  建模软件统一采用 3DMAX，推荐使用 3D MAX 2023 版本及以上。（版本自带GLB格式导出）
2. 软件使用之前需要把单位设置为“CM”。

#### 建模过程

1.  在模型制作中，如果需要一个模型放在，或者紧贴另一个模型时，请使用捕捉吸附工具（快捷键S）， 避免有些不合理的穿帮问题。
2.  模型坐标轴请调整至模型最中心，Z轴调整为模型最底部。如果觉得调整比较麻烦，可以在顶视图新建物体，这样Z轴会自动在物体的最底部，X、Y则在物体中心。

#### 模型命名规范

1.  模型，贴图的命名一定不能有重名。
2.  模型命名依据使用情况用中文或字母开头，不能用数字开头。
3.  名称不能出现特殊符号如&、*、空格一类，间隔使用下划线“_”。
4.  如果同类模型较多则命名规则为“墙_01、墙_02或mur_01、mur_02”。

#### 模型面要求

1.  模型不能有多余的点、线、面。
2.  模型接边处不能有缝隙，穿插的模型多余的面，看不到的面需要删除。
3.  不能有漏面，重面，破面等。
4.  检查模型有没有黑面，漏面的等情况。

#### 贴图标准

1.  贴图尽量为1：1正方形，且像素值为 2 的 n 次方。贴图大小范围控制在32X32—2048X2048。
2.  贴图的文件格式为：jpg、png。
3.  使用元素重复度高的贴图时，务必只裁切其中一部分并在 3DMAX 中把模型 UV 平铺且缩放致所需效果。严禁把贴图复制，制作成大贴图赋予模型。
4.   所有模型必须有贴图，如果模型是纯色，也必须用颜色贴图赋予。不能在 3DMAX 里面用材质球赋予，如果这样导出模型后在VC Hub环境里使用颜色会变淡，不符合模型原来的颜色。
5.  贴图命名则对应模型名称，如果一种模由多种材质组成后缀用“_01、_02区分”。

# 动作

通过动作触发系统中的其他操作或程序，以响应该动作并执行相应任务。

WAGO VC Hub中，在画面或者控件的”动作“栏中设置具体动作。

![alt text](1.png)

## 如何触发动作

可以通过以下操作触发特定的动作。

#### 鼠标事件

所有控件均支持鼠标事件。

鼠标事件是指当用户使用鼠标对画面或者控件进行操作时，触发特定的动作。包含：

- 鼠标按下
- 鼠标抬起
- 鼠标进入
- 鼠标离开

#### 属性变化

当控件的属性值（目前仅支持值属性）发生变化时，触发特定的动作。

以下控件才支持属性变化事件：

- 下拉框
- 文本输入框
- 滑动条
- 日历
- 数字输入框
- 文本标签
- 值显示
- 数码管
- 复选框
- 单选按钮

#### 生命周期

所有控件均支持生命周期标事件。

当画面或者控件被加载或者销毁时，触发特定的动作。

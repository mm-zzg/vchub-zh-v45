# 分辨率自适应

分辨率自适应可以使网页在不同的设备上正常运行。无论是在桌面电脑、平板还是手机等移动设备上，用户都可以轻松地访问。

**场景举例一：** 在不同的设备上访问同一个画面。

1. 绘制画面，并且开启分辨率自适应，选择宽高等比缩放。

    ![alt text](42.png)

2. 分别在电脑、平板和手机上查看该画面的运行页面。

    ![alt text](43.png)
